3º ano C

  • Laços de repetição: Estruturas de controle de fluxo que permitem executar um bloco de código repetidamente enquanto uma condição for verdadeira, como o for, while e do-while.
  • Linguagem de programação: Conjunto de regras e sintaxes usadas para escrever programas que instruem um computador a realizar tarefas específicas. Exemplos incluem Python, Java, JavaScript, C, entre outros.
  • Principais TAGs utilizadas: Tags HTML que são comumente usadas para criar páginas web, como <div>, <p>, <a>, <h1>, <ul>, etc.
  • Background-Color: Propriedade CSS usada para definir a cor de fundo de um elemento HTML, como background-color: #ff0000;.
  • Font-Size: Propriedade CSS usada para definir o tamanho da fonte de um texto em um elemento HTML, como font-size: 16px;.
  • Text-Align: Propriedade CSS usada para alinhar o texto dentro de um elemento HTML. Pode ser left, right, center, ou justify.
  • Text-Transform: Propriedade CSS usada para modificar a aparência do texto, como uppercase, lowercase, capitalize, entre outras.
  • Console (o que é e para que serve): Ferramenta de desenvolvimento usada para exibir mensagens de erro, logs ou interagir com o código em execução, geralmente encontrada em navegadores ou ambientes de desenvolvimento como o Node.js.

A tag input é usada para criar controles interativos que permitem ao usuário inserir dados. Ela é uma tag vazia (não possui conteúdo interno) e seu comportamento e aparência são determinados principalmente pelo atributo type.

A tag label é usada para associar um rótulo (texto descritivo) a um elemento de formulário, geralmente um input. Essa associação é crucial para a usabilidade e acessibilidade.
